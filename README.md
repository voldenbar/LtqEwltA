# 项目简介

## 核心功能
- 基于YOLOv8的木材表面缺陷检测，支持识别Crack、Dead Knot、Live Knot三类缺陷
- 提供2606张标注数据集
- 支持图片、视频及摄像头三种输入方式进行检测
- 支持批量检测功能，可在界面直接查看所有检测结果
- 支持检测结果保存
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 可选择不同的检测模型

## 适用场景
- 木材加工行业的表面缺陷自动化检测
- 木材质量控制和分类
- 生产线实时监控
- 木材质检报告生成

## 技术特点
- 基于Python开发，使用Pycharm+Anaconda或VSCode+Anaconda作为开发环境
- 采用PyQt5构建用户界面
- 依赖opencv-python、ultralytics、torch1.9等深度学习框架
- 提供完整的源码、数据集、训练模型及运行教程
- 已训练好模型，配置成功环境可直接使用
